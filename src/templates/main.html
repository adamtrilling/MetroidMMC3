Metroid Randomizer

Fill in a Seed to download a randomized version of Metroid that should be playable in an emulator.

Guarantees

  • The same seed will generate the same output, so that the Randomizer can be used for races
  • The Morph Ball is in its normal location because you can't go a lot of places without it

Caveats

  • I think every game is winnable, but I can't prove it. If you have the Morph Ball, Missiles, and do the right screen wraps, you should be able to get to any location on the map. You can get Missiles with just the Morph Ball by defeating Kraid.
  • This is not anywhere close to my usual standards of code quality. If it turns out to work and be enjoyable for people, I'll put a lot more time into it. But expect bugs.
